# 图示 (Les schémas et graphes)

## L图式 (Le schéma L)

```
    (Es) S ──────────────── a' (autre)
           \              /
            \            /
             \          /
              \        /
               \      /
            imaginaire
               /      \
              /        \
             /          \
            /            \
           /              \
    (moi) a ──────────────── A (Autre)
               symbolique
```

**四个位置**：
- **S (Es)**：主体，但是是无意识的主体，弗洛伊德的 Es（它）
- **a (moi)**：自我，想象性的我
- **a' (autre)**：小他者，镜像，相似者
- **A (Autre)**：大他者，象征秩序

**两条轴**：
- **想象轴 (a—a')**：自我与镜像的关系，想象性认同
- **象征轴 (S—A)**：主体与大他者的关系，被想象轴阻断

**关键洞见**：
- 想象轴**阻断**了象征轴
- 自我与小他者的关系遮蔽了主体与大他者的关系
- 这就是为什么"自我"是分析的障碍，而非盟友
- 分析要做的：绕过想象轴，让 S—A 的关系显现

"你以为你在和我说话，其实你在和你的镜像说话。"

## 欲望图 (Le graphe du désir)

### 第一层：基本图

```
        Δ ───────────────→ $
         \                ↑
          \              /
           \   A    s(A)/
            ↘   ↓   ↗
             ───●───
            ↗       ↘
           /         \
          /           \
        意向 ←───────── 能指链
```

**元素**：
- **Δ**：神话性的起点，未被象征化的需要
- **$**：被划杠的主体，言说的结果
- **A**：大他者，语言的宝库
- **s(A)**：大他者的所指，意义的效果
- **能指链**：S₁ → S₂ → S₃ ...

**过程**：
- 主体从 Δ 出发，带着需要
- 穿过能指链，需要被语言切割
- 到达 A（大他者）寻求意义
- 但得到的只是 s(A)——总是不完整的意义
- 主体作为 $ 出现——被划杠的，分裂的

### 第二层：完整图

```
        享乐            Che vuoi?
          ↑               ↑
    $ ◇ a ←─── $ ◇ D ←─── d
          ↓               ↓
        S(Ⱥ)   ────────  s(A)
                A
```

**上层新增元素**：
- **d**：欲望
- **$ ◇ D**：欲望的公式（主体对要求的关系）
- **$ ◇ a**：幻想的公式
- **S(Ⱥ)**：被划杠的大他者的能指——大他者也是不完整的

**Che vuoi?**（你想要什么？）：
- 大他者向主体提出的问题
- 主体无法回答，因为他不知道自己的欲望
- 这个问题制造焦虑
- 主体用幻想（$ ◇ a）来回答这个无法回答的问题

**欲望的辩证法**：
- 需要（besoin）→ 被语言切割
- 要求（demande）→ 对爱的要求，超出需要
- 欲望（désir）→ 在需要和要求的差距中诞生
- 欲望是要求减去需要后的剩余

"你要求的不是你需要的，你欲望的不是你要求的。"

## 四种话语的公式

每种话语都有四个位置：

```
施动者 → 他者
  ↑        ↓
真理    产品
```

### 主人话语 (Discours du maître)

```
S₁ → S₂
──   ──
$     a
```

- **施动者**：主人能指 S₁（命令）
- **他者**：知识 S₂（奴隶的知识）
- **真理**：被划杠的主体 $（主人的分裂被掩盖）
- **产品**：客体小a（剩余享乐，主人得不到的）

主人说"做！"，奴隶知道怎么做，但产出的享乐主人无法获得。

### 大学话语 (Discours de l'université)

```
S₂ → a
──   ──
S₁   $
```

- **施动者**：知识 S₂（"客观的"知识）
- **他者**：学生作为客体 a
- **真理**：隐藏的主人能指 S₁（权威）
- **产品**：被划杠的主体 $（被生产出的顺从主体）

大学话语的虚伪：表面上是知识在说话，实际上背后是权力。

### 癔症话语 (Discours de l'hystérique)

```
$ → S₁
──   ──
a    S₂
```

- **施动者**：被划杠的主体 $（分裂的、提问的）
- **他者**：主人能指 S₁（被质问的权威）
- **真理**：客体小a（欲望的秘密原因）
- **产品**：知识 S₂（主人被迫生产知识）

癔症者的提问迫使主人暴露他的无知。分析者应促成癔症话语。

### 分析师话语 (Discours de l'analyste)

```
a → $
──   ──
S₂   S₁
```

- **施动者**：客体小a（分析师占据欲望客体-原因的位置）
- **他者**：被划杠的主体 $（分析者）
- **真理**：知识 S₂（分析师的知识，但不直接运用）
- **产品**：主人能指 S₁（分析者生产出自己的主人能指）

分析师的沉默、切断、谜一般的在场——都是为了占据 a 的位置，让分析者生产出他自己的 S₁。

"分析师不是知道答案的人，而是让你生产出自己答案的人。"
